Bienvenue sur Chatbot-AI

Description

Chatbot-AI est une plateforme de messagerie intelligente exploitant des LLM en local via Ollama. Elle offre une gestion complète de l'historique et une sécurité renforcée via Keycloak.

Technologies

  • Frontend : Angular 21, Bootstrap
  • Backend : Node.js (Express)
  • Infra & DB : Docker, Keycloak, Ollama, Prisma
